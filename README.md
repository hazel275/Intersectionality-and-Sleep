# Intersectionality and Sleep - Analysis Code

Code for the intersectional sleep-duration analysis (All of Us CDR v8, BYOD Fitbit).

## Data statement
This repository contains analysis CODE only and NO All of Us participant-level data.
Any aggregate results follow the All of Us Data and Statistics Dissemination Policy:
no reported cell corresponds to fewer than 20 participants. Reproduce results inside
the All of Us Researcher Workbench.
